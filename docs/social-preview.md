# Social Preview

GitHub social preview image for `techlogist1/flint`.

## Specs

- **Dimensions:** 1280 × 640 px
- **Background:** `#050505` (Flint's `--bg-void`)
- **Foreground:** `flintcolorlogo.png` centered, roughly 60% width
- **Optional tagline:** "Enter the flint state." in JetBrains Mono below the logo, `#16a34a` (Flint's `--accent`)
- **Output:** `docs/social-preview.png`

## Where it's used

Upload via GitHub → Settings → Social preview → Upload an image. GitHub serves this when the repo is linked on Twitter, Slack, Discord, etc. Replaces the default "owner/name" card.
